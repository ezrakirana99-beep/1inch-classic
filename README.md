# 1inch Classic (Personal)

Static frontend that replicates 1inch classic UI for personal use.
Features: Wallet connect (MetaMask), simple swap stub, limit order fill form.

## How to use
1. Place ABI of 1inch LimitOrder/Aggregation Router into `abi/aggregationRouterV2.json`.
2. Enable GitHub Pages (Settings → Pages → Branch `main` → /root).
3. Visit the published site.
4. Connect MetaMask and interact (approve/take actions) on chosen network.

**Security:** This is a frontend only. Never store private keys in repo.
